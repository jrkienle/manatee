# Basic Tutorial

This tutorial will teach you how to create an incredibly basic 2D game with Manatee!

## Assumptions

This tutorial assumes the following:

1. You've followed all setup instructions
2. You have a basic understanding of the Rust programming language
3. You have a basic understanding of how video games work

With that in mind, let's begin!

## Creating and Running the Game Loop

The first thing that we'll need to do is create and then start our game loop. Let's start off by
opening our game's `main.rs` and importing `Game` from `manatee`:

```rust
use manatee::Game, Scene;

pub fn main() {
  // This should be empty (for now)
}
```

That was easy!
